# tools
Assorted useful tools, almost entirely generated using LLMs.
Inspired by https://github.com/simonw/tools

Live at **https://tools.chenna.me**

## Tools

- [Plist Editor](plist-editor.html) — Upload an XML plist file to view and interactively edit its structure as an expandable tree.

## Building the index

`index.html` is generated by `build_index.py`. Run it whenever tools are added:

```bash
python build_index.py
```

The script scans every `*.html` file (excluding `index.html`), extracts its
`<title>`, reads the matching `*.docs.md` for a description, and writes
`index.html` with embedded JSON and a client-side real-time search UI.
